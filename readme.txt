Hello Udacity | AWS Static Website Project
About The Project

This project demonstrates hosting a responsive static website on AWS S3 and distributing it globally via CloudFront CDN. The site is mobile-friendly, professionally styled, and helps learners understand AWS static website deployment, permissions, and content delivery optimization.

Live Website: http://hello-udacity-adithya.s3-website-us-east-1.amazonaws.com

CloudFront URL: https://d3pblbe9k639wp.cloudfront.net

Project Highlights

✅ Static website hosted on AWS S3

✅ CloudFront CDN for faster global access

✅ Responsive and mobile-friendly design

✅ Modern professional styling

✅ Full website and distribution tested successfully

Getting Started
Prerequisites

AWS account with S3 and CloudFront access

Basic understanding of HTML and CSS

Steps Performed

Created S3 bucket: hello-udacity-adithya in US East (N. Virginia)

Enabled Static Website Hosting

Uploaded website files (index.html & style.css)

Set Bucket Policy for public-read access

Created CloudFront Distribution for HTTPS and global access

Tested the website using both S3 and CloudFront URLs

Usage

Open the CloudFront URL in any browser to see the live website.
Use this as a reference for deploying static websites, configuring permissions, and distributing content via AWS CloudFront.
